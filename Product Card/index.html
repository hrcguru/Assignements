Dynamic Product Cards

Create beautiful product cards with custom properties

How to use this application:

  • Fill out the form with product details
  • Name must be a non-empty string
  • Price must be a positive number
  • Image URL must be a valid URL starting with "http"
  • Discount (optional) must be between 0 and 100
  • Click "Add Product" to create a new product card
