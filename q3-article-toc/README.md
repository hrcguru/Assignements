# Article Layout with Sticky Table of Contents

A responsive article layout featuring a main content area and a sticky table of contents sidebar, built with semantic HTML and CSS Grid.

## Features

- **Semantic HTML5** structure using `<main>`, `<article>`, `<nav>`, and proper heading hierarchy
- **CSS Grid layout** for two-column desktop view
- **Sticky Table of Contents** that remains visible while scrolling
- **Fully responsive** - switches to single column on mobile devices
- **Accessibility features**:
  - Skip navigation link
  - Proper focus indicators
  - Semantic landmark elements
  - ARIA labels
- **Smooth scrolling** for anchor links
- **No JavaScript required**
