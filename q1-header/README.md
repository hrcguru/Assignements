# Responsive Header (Q1)

A responsive, sticky header built with Flexbox and CSS positioning.

## Features

- **Sticky Header**: Remains visible at the top when scrolling
- **Flexbox Layout**: Logo (left), navigation (center), CTA button (right)
- **Responsive Design**: Adapts gracefully from desktop to mobile
- **Accessibility**: Proper focus states and keyboard navigation
- **Semantic HTML**: Uses appropriate HTML5 elements

## How to View

1. Clone the repository or download the files
2. Open `index.html` in a web browser
3. Scroll down to see the sticky header behavior
4. Resize the browser window to test responsive behavior
